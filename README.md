# Monte Carlo Portfolio Risk Simulator

A Python package that simulates correlated multi-asset portfolio paths and computes risk metrics (VaR/CVaR), with Matplotlib visualizations.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # mac/linux
# .venv\Scripts\activate   # windows

pip install -U pip
pip install -e ".[dev]"
